Framework
----------
A framework is a set of conceptual structure and guidelines, used to build something useful.
It may include predefined classes and functions that can be used to process input, manage hardware devices, and interact with system software.
The purpose of the framework is to allow developers to focus on building a unique features for their Projects rather than writing code from scratch.

Why use Framework?
- collection of tools
- no need to start from scratch
- save time
- improve productivity
- clean code
- reusable code
- testing 
- debugging
